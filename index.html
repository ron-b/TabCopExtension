Privacy Information

Tab Cop doesn't collect personal data, browsing history, open tabs, etc - nor it ever will.

We use Chrome Sync Storage API to store the user's settings across different sessions and different devices.

We use Google Analytics to help improve our product by sending basic usage metrics to understand what features are used / should be removed, you can opt out of sending analyitics in the options page.
